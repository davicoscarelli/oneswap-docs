## 1. Install

```bash
npm install @oneswap/sdk
```

## 2. Get your API key

Sign up at [oneswap.cc](https://oneswap.cc) and grab an API key from your dashboard.

## 3. Initialize

```typescript
import { OneSwap } from '@oneswap/sdk'

const client = new OneSwap({ apiKey: 'os_live_...' })
```

## 4. Create a wallet for your user

```typescript
const wallet = await client.wallets.create({ walletAddress: 'alice::12205a8c...' })
// wallet.address â†’ Canton party ID, managed for you
```

## 5. Get a quote

```typescript
const quote = await client.quotes.get({
  from: 'Amulet',
  to: 'USDCx',
  amount: '100'
})

console.log(quote.outputAmount)              // "48.392810"
console.log(quote.rate)                      // "0.483928"
console.log(quote.priceImpact)               // "0.12"
console.log(quote.estimatedTrafficCost?.cc)  // "0.012345" (network fee in CC)
console.log(quote.inputAmountAfterTrafficFee) // "99.987655" (net input)
```

## 6. Execute a swap

```typescript
const swap = await client.swaps.create({
  fromToken: 'Amulet',
  toToken: 'USDCx',
  amount: '100',
  walletAddress: 'alice::12205a8c...'
})

console.log(swap.depositAddress) // where tokens need to go
console.log(swap.expectedOutput) // what you'll get back
console.log(swap.expiresAt)      // 30 min to deposit
```

## 7. Wait for completion

```typescript
swap.on('matched', () => console.log('Deposit detected'))

const result = await swap.wait()
console.log(`Done! Got ${result.actualOutput} ${result.outputToken}`)
```

If slippage is exceeded, the swap is refunded automatically.

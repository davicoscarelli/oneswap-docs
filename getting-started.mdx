## Install

```bash
npm install @oneswap/sdk
```

## Get an API key

1. Register a developer account:

```bash
curl -X POST https://api.oneswap.cc/api/sdk/register \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "password": "..."}'
```

2. Log in to get a JWT:

```bash
curl -X POST https://api.oneswap.cc/api/sdk/login \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "password": "..."}'
```

3. Create an API key:

```bash
curl -X POST https://api.oneswap.cc/api/sdk/keys \
  -H "Authorization: Bearer <jwt>"
```

## Initialize the client

```typescript
import { OneSwap } from '@oneswap/sdk'

const client = new OneSwap({
  apiKey: 'os_live_...',
  baseUrl: 'https://api.oneswap.cc', // optional, this is the default
  timeout: 30000                      // optional, milliseconds
})
```

## Verify it works

```typescript
const pools = await client.pools.list()
console.log(`Found ${pools.length} pools`)
```
